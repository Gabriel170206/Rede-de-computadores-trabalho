# Envio de IP de PC0 para PC1

Neste cenário do Packet Tracer, PC0 envia um pacote ICMP para PC1 usando o comando de ping.

## Detalhes observados

- Comando executado: `ping 192.168.10.11`
- Resposta recebida de `192.168.10.11` com `bytes=32`, `time=4ms` e `TTL=128`
- Total de pacotes: 4 enviados, 4 recebidos, 0 perdidos
- Estatísticas de tempo:
  - Mínimo = 4ms
  - Máximo = 11ms
  - Média = 5ms

## Conclusão

PC0 alcançou PC1 com sucesso no cenário. O ping indicou que a comunicação IP entre os dois dispositivos está funcionando, sem perda de pacotes.
